# Проект «Музей Победы»
### Информационный веб-ресурс, посвященный истории Великой Отечественной войны

## Описание проекта
Сайт представляет собой образовательный портал о музейных экспозициях, героях и ключевых событиях истории. Проект направлен на сохранение исторической памяти и патриотическое воспитание молодежи.

## Реализованная структура:
*   `index.html` — Главная страница музея (входная точка).
*   `register.html` / `login.html` — Функционал регистрации и авторизации.
*   `profile.html` — Личный кабинет пользователя.
*   `/css/` — Стилизация интерфейса (Native CSS3).

## Технологии и особенности:
*   **HTML5 / CSS3** — Чистая семантическая верстка.
*   **Медиа-контент** — Интеграция видео и графических материалов.
*   **GIT** — Контроль версий и ведение истории разработки.
